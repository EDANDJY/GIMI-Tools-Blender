# GIMI-Blender-Plugin 功能对照表

## 原 GIMI-Tools 脚本功能 vs Blender 插件功能

| 原脚本文件 | 命令行用法 | Blender 插件功能 | 插件中的位置 |
|-----------|-----------|----------------|-------------|
| **genshin_3dmigoto_collect.py** | `python genshin_3dmigoto_collect.py -vb <hash> -n <name> ...` | **从帧分析提取对象** | 模式：导入对象 |
| **genshin_3dmigoto_generate.py** | `python genshin_3dmigoto_generate.py -n <name> ...` | **导出 Mod** | 模式：导出 Mod |
| **genshin_animation_creator.py** | `python genshin_animation_creator.py -n <vg_count> -k <key> ...` | **创建动画模组** | 模式：动画创建 |
| **genshin_merge_mods.py** | `python genshin_merge_mods.py -k <key> -c -ref ...` | **合并模组** | 模式：合并模组 |
| **genshin_auto_damage_merge.py** | `python genshin_auto_damage_merge.py -k <key> -acc -hit ...` | **创建自动伤害模组** | 模式：自动伤害切换 |
| **genshin_set_outlines.py** | `python genshin_set_outlines.py --thickness <0-255>` | **设置轮廓厚度** | 模式：工具箱 > 轮廓工具 |
| **genshin_set_color.py** | `python genshin_set_color.py -n <file> -r -g -b -a ...` | **设置颜色** | 模式：工具箱 > 颜色工具 |
| **genshin_set_transparency.py** | `python genshin_set_transparency.py -b -hd -d -e -t <r> <g> <b>` | **设置透明度** | 模式：工具箱 > 透明度工具 |

## 详细功能映射

### 1. 导入对象 (genshin_3dmigoto_collect.py)

**原命令行参数** → **Blender UI 选项**

- `-vb` (VB 哈希) → `指定哈希` 输入框
- `-n` (名称) → 自动使用 "Character"
- `-f` (帧分析目录) → `帧分析目录` 路径选择器
- `--force` (强制收集) → 未来版本支持
- `--has_blend` (强制混合数据) → 自动检测
- `--no_blend` (无混合数据) → 自动检测
- `--has_normalmap` (有法线贴图) → 自动检测
