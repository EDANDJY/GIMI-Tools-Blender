# GIMI-Blender-Plugin 功能对照表

## 原 GIMI-Tools 脚本功能 vs Blender 插件功能

| 原脚本文件 | 命令行用法 | Blender 插件功能 | 插件中的位置 |
|-----------|-----------|----------------|-------------|
| **genshin_3dmigoto_collect.py** | `python genshin_3dmigoto_collect.py -vb <hash> -n <name> ...` | **从帧分析提取对象** | 模式：导入对象 |
| **genshin_3dmigoto_generate.py** | `python genshin_3dmigoto_generate.py -n <name> ...` | **导出 Mod** | 模式：导出 Mod |
| **genshin_animation_creator.py** | `python genshin_animation_creator.py -n <vg_count> -k <key> ...` | **创建动画模组** | 模式：动画创建 |
| **genshin_merge_mods.py** | `python genshin_merge_mods.py -k <key> -c -ref ...` | **合并模组** | 模式：合并模组 |
| **genshin_auto_damage_merge.py** | `python genshin_auto_damage_merge.py -k <key> -acc -hit ...` | **创建自动伤害模组** | 模式：自动伤害切换 |
| **genshin_set_outlines.py** | `python genshin_set_outlines.py --thickness <0-255>` | **设置轮廓厚度** | 模式：工具箱 > 轮廓工具 |
| **genshin_set_color.py** | `python genshin_set_color.py -n <file> -r -g -b -a ...` | **设置颜色** | 模式：工具箱 > 颜色工具 |
| **genshin_set_transparency.py** | `python genshin_set_transparency.py -b -hd -d -e -t <r> <g> <b>` | **设置透明度** | 模式：工具箱 > 透明度工具 |

## 详细功能映射

### 1. 导入对象 (genshin_3dmigoto_collect.py)

**原命令行参数** → **Blender UI 选项**

- `-vb` (VB 哈希) → `指定哈希` 输入框
- `-n` (名称) → 自动使用 "Character"
- `-f` (帧分析目录) → `帧分析目录` 路径选择器
- `--force` (强制收集) → 未来版本支持
- `--has_blend` (强制混合数据) → 自动检测
- `--no_blend` (无混合数据) → 自动检测
- `--has_normalmap` (有法线贴图) → 自动检测

**UI 选项**:
- [x] 跳过小尺寸贴图
  - 最小尺寸 (KB): 256
- [x] 跳过 .jpg
- [x] 跳过已知立方体贴图
- [ ] 跳过同槽哈希贴图
- [x] 跳过空顶点组
- [ ] 镜像网格
- 顶点颜色：线性/非线性
- 骨架类型：合并型/组件型

### 2. 导出 Mod (genshin_3dmigoto_generate.py)

**原命令行参数** → **Blender UI 选项**

- `-n` (名称) → `Mod 名称`
- `--hashfile` (哈希文件) → 自动使用 hash.json
- `--original_tangents` (原始切线) → `使用原始切线` 复选框

**UI 选项**:
- 组件集合：集合/多个
- LODO 来源目录：路径选择器
- Mod 导出目录：路径选择器
- 骨架类型：合并型/组件型
- [ ] 镜像网格
- [ ] 应用全部修改器
- [x] 复制贴图
- [x] 更新贴图
- [x] 生成 Mod INI
- [ ] 为 INI 添加注释
- [x] 忽略嵌套集合
- [ ] 忽略隐藏对象
- [x] 忽略已禁用形态键
- [x] 跳过已知立方体贴图
- [x] 补全缺失顶点组
- [ ] 不限制自定义形态键
- 骨架缩放：1.00
- [ ] 部分导出

**Mod 信息**:
- Mod 名称
- 作者名称
- Mod 描述
- Mod 链接
- Mod Logo

### 3. 动画创建 (genshin_animation_creator.py)

**原命令行参数** → **Blender UI 选项**

- `-r` (根目录) → 当前目录
- `-s` (存储) → 未来版本支持
- `-e` (启用) → 未来版本支持
- `-n` (名称) → 自动使用 anim.ini
- `-k` (按键) → `切换键` 输入框
- `-a` (仅活动) → `仅活动角色` 复选框
- `-v` (顶点组) → `顶点组数量` 输入框

### 4. 合并模组 (genshin_merge_mods.py)

**原命令行参数** → **Blender UI 选项**

- `-r` (根目录) → 当前目录
- `-s` (存储) → 未来版本支持
- `-e` (启用) → 未来版本支持
- `-n` (名称) → 自动使用 merged.ini
- `-k` (按键) → `切换键` 输入框
- `-c` (压缩) → `压缩输出` 复选框
- `-a` (仅活动) → `仅活动角色` 复选框
- `-ref` (反射修复) → `反射修复` 复选框

### 5. 自动伤害切换 (genshin_auto_damage_merge.py)

**原命令行参数** → **Blender UI 选项**

- `-r` (根目录) → 当前目录
- `-s` (存储) → 未来版本支持
- `-e` (启用) → 未来版本支持
- `-n` (名称) → 自动使用 auto.ini
- `-k` (按键) → `切换键` 输入框
- `-c` (压缩) → `压缩输出` 复选框
- `-a` (仅活动) → `仅活动角色` 复选框
- `-ref` (反射修复) → `反射修复` 复选框
- `-acc` (累积) → `伤害累积` 复选框
- `-hit` (单次伤害) → `单次大伤害` 复选框

### 6. 轮廓工具 (genshin_set_outlines.py)

**原命令行参数** → **Blender UI 选项**

- `--thickness` (厚度 0-255) → `轮廓厚度` 滑块 (0-255)

### 7. 颜色工具 (genshin_set_color.py)

**原命令行参数** → **Blender UI 选项**

- `-n` (文件名) → 自动检测 Texcoord.buf
- `-r` (红色值) → `R` 输入框 (0-255)
- `-g` (绿色值) → `G` 输入框 (0-255)
- `-b` (蓝色值) → `B` 输入框 (0-255)
- `-a` (透明度值) → `A` 输入框 (0-255)
- `--stride` (步长) → 默认 20（未来可配置）

### 8. 透明度工具 (genshin_set_transparency.py)

**原命令行参数** → **Blender UI 选项**

- `-b` (身体) → `身体透明` 复选框
- `-hd` (头部) → `头部透明` 复选框
- `-d` (衣服) → `衣服透明` 复选框
- `-e` (额外) → `额外透明` 复选框
- `-t` (透明度因子 RGB) → `R/G/B 因子` 滑块 (0.0-1.0)

## 新增功能

相比原 GIMI-Tools，Blender 插件新增了以下功能：

1. **图形化界面** - 无需记忆命令行参数
2. **中英文切换** - 支持多语言界面
3. **参数验证** - 自动验证输入路径和参数
4. **错误提示** - 友好的错误消息显示
5. **集成工作流** - 所有功能在一个界面中完成
6. **实时反馈** - 操作结果即时显示在 Blender 中

## 使用优势

### 原 GIMI-Tools（命令行）
```bash
# 需要打开终端
cd GIMI-Tools
python genshin_3dmigoto_collect.py -vb 12345678 -n Character -f ./FrameAnalysis
# 需要记住所有参数
python genshin_set_outlines.py --thickness 0
```

### GIMI-Blender-Plugin（图形界面）
```
1. 打开 Blender
2. 按 N 键打开侧边栏
3. 点击 GIMI Tools 标签
4. 选择"导入对象"模式
5. 点击文件夹图标选择路径
6. 点击"从帧分析提取对象"按钮
```

## 兼容性说明

- ✅ 完全兼容原 GIMI-Tools 的所有功能
- ✅ 输出文件格式完全相同
- ✅ 支持所有原命令行参数（通过 UI 配置）
- ⚠️ 需要 GIMI-Tools 脚本文件存在（插件调用这些脚本）
- ⚠️ Blender 3.6.0 或更高版本
