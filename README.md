# GIMI-Blender-Plugin

原神 (Genshin Impact) 模组开发 Blender 插件 - 基于 SilentNightSound 的 GIMI-Tools

## 功能特性

本插件将原 GIMI-Tools 的命令行工具集成到 Blender 中，提供图形化界面，无需在终端输入命令即可使用所有功能。

### 主要功能

1. **导入对象** - 从 3DMigoto 帧分析数据中提取并导入模型
2. **导出 Mod** - 将 Blender 场景导出为 Genshin Impact Mod
3. **动画创建** - 创建可切换的动画模组
4. **合并模组** - 将多个模组合并为一个可切换的模组
5. **自动伤害切换** - 根据伤害值自动切换服装/模组
6. **工具箱** - 各种实用工具
   - 设置轮廓厚度
   - 设置顶点颜色
   - 设置透明度效果

### 界面语言

- 支持中文和英文界面
- 可在插件面板顶部切换语言

## 安装方法

### 前提条件

1. Blender 3.6.0 或更高版本
2. Python 3.x（Blender 自带）
3. GIMI-Tools 脚本文件（位于 `GIMI-Tools` 文件夹）

### 安装步骤

1. **准备文件**
   - 确保 `GIMI-Blender-Plugin` 文件夹和 `GIMI-Tools` 文件夹在同一目录下
   - 目录结构应该如下：
     ```
     Tools/
     ├── GIMI-Blender-Plugin/     # 插件文件夹
     │   ├── __init__.py
     │   ├── auto_load.py
     │   ├── addon/
     │   ├── language/
     │   ── ...
     └── GIMI-Tools/              # GIMI 脚本文件夹
         ├── blender_3dmigoto_gimi.py
         ├── genshin_3dmigoto_collect.py
         ├── genshin_3dmigoto_generate.py
         └── ...
     ```

2. **在 Blender 中安装**
   - 打开 Blender
   - 进入 `编辑` > `偏好设置` > `插件`
   - 点击 `安装...` 按钮
   - 选择 `GIMI-Blender-Plugin` 文件夹
   - 勾选插件启用

3. **访问插件**
   - 在 3D 视图窗口中，按 `N` 键打开侧边栏
   - 点击 `GIMI Tools` 标签页

## 使用说明

### 1. 导入对象

从帧分析数据导入模型：

1. 选择模式为"导入对象"
2. 设置帧分析目录（FrameAnalysis 文件夹路径）
3. 设置提取输出目录
4. （可选）指定 VB 哈希值
5. 配置导入选项：
   - 跳过小尺寸贴图
   - 跳过 JPG 格式
   - 跳过立方体贴图
   - 镜像网格
   - 顶点颜色格式
   - 骨架类型
6. 点击"从帧分析提取对象"按钮

### 2. 导出 Mod

将场景导出为 Mod：

1. 选择模式为"导出 Mod"
2. 设置 LODO 来源目录
3. 设置 Mod 导出目录
4. 配置导出选项：
   - 骨架类型
   - 镜像网格
   - 应用修改器
   - 复制/更新贴图
   - 生成 INI 文件
   - 补全缺失顶点组
   - 骨架缩放
5. 填写 Mod 信息：
   - Mod 名称
   - 作者名称
   - Mod 描述
   - Mod 链接
   - Mod Logo
6. 点击"导出 Mod"按钮

### 3. 创建动画模组

1. 选择模式为"动画创建"
2. 设置顶点组数量
3. 设置切换键（单个字母）
4. 选择是否仅活动角色
5. 点击"创建动画模组"按钮

### 4. 合并模组

1. 选择模式为"合并模组"
2. 将多个 Mod 文件夹放在同一目录下
3. 设置切换键
4. 配置选项：
   - 压缩输出（使文件更小，但难以还原）
   - 反射修复（3.0+ 角色）
5. 点击"合并模组"按钮

### 5. 自动伤害切换

1. 选择模式为"自动伤害切换"
2. 将多个 Mod 文件夹放在同一目录下
3. 配置选项：
   - 伤害累积模式
   - 单次大伤害模式
   - 切换键
   - 压缩输出
4. 点击"创建自动伤害模组"按钮

### 6. 工具箱

#### 设置轮廓厚度

1. 选择模式为"工具箱"
2. 在"轮廓工具"部分设置厚度（0-255）
   - 0 = 无轮廓
   - 255 = 最厚轮廓
3. 点击"应用轮廓厚度"按钮

#### 设置颜色

1. 在"颜色工具"部分设置 RGBA 值（0-255）
2. 点击"应用颜色"按钮
   - 常用于移除轮廓（设置 A=0）

#### 设置透明度

1. 在"透明度工具"部分选择要透明的身体部位
2. 设置 RGB 透明度因子（0.0-1.0）
3. 点击"应用透明度"按钮

## 注意事项

1. **路径配置**
   - 确保 GIMI-Tools 脚本文件夹与插件文件夹在同一目录
   - 所有路径应使用绝对路径

2. **依赖项**
   - 插件依赖原始的 GIMI-Tools Python 脚本
   - 确保所有 `.py` 文件都在 `GIMI-Tools` 文件夹中

3. **兼容性**
   - 适用于 Blender 3.6.0+
   - 需要 Python 3.x

4. **错误处理**
   - 如果操作失败，查看 Blender 系统控制台（窗口类型 > 系统控制台）
   - 错误信息会显示在报告区域

## 故障排除

### 插件无法加载

- 检查 Blender 版本是否为 3.6.0 或更高
- 确认所有必要的 Python 文件都存在
- 查看 Blender 控制台错误信息

### 操作符执行失败

- 确保 GIMI-Tools 文件夹在正确位置
- 检查输入路径是否正确
- 查看系统控制台的详细错误信息

### 导入/导出问题

- 验证帧分析文件夹是否包含正确的文件
- 检查输出目录是否有写入权限
- 确认模型没有拓扑错误
