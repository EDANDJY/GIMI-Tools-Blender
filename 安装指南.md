## 安装步骤

### 方法 1：在 Blender 中直接安装

1. 打开 Blender
2. 进入 `编辑 (Edit)` > `偏好设置 (Preferences)` > `插件 (Add-ons)`
3. 点击 `安装... (Install...)` 按钮
4. 导航到 `GIMI-Blender-Plugin` 文件夹
5. 选择该文件夹并点击 `安装插件 (Install Add-on)`
6. 勾选插件名称旁边的复选框以启用它
7. 点击 `保存偏好设置 (Save Preferences)`

### 方法 2：手动复制

1. 找到 Blender 的插件目录：
   - Windows: `C:\Users\<用户名>\AppData\Roaming\Blender Foundation\Blender\<版本>\scripts\addons\`
   - macOS: `/Users/<用户名>/Library/Application Support/Blender/<版本>/scripts/addons/`
   - Linux: `/home/<用户名>/.config/blender/<版本>/scripts/addons/`

2. 将整个 `GIMI-Blender-Plugin` 文件夹复制到该目录

3. 打开 Blender，进入 `编辑` > `偏好设置` > `插件`

4. 搜索 "GIMI" 并勾选启用

5. 点击 `保存偏好设置`

## 使用插件

1. 在 Blender 中，打开 3D 视图窗口
2. 按 `N` 键打开右侧边栏
3. 点击 `GIMI Tools` 标签
4. 选择语言（中文/英文）
5. 选择需要的功能模式

## 功能说明

### 1. 导入对象模式
从 3DMigoto 帧分析数据导入模型到 Blender

### 2. 导出 Mod 模式
将 Blender 中的模型导出为 Genshin Impact Mod

### 3. 动画创建模式
创建可切换的动画模组

### 4. 合并模组模式
将多个模组合并为一个可切换的模组

### 5. 自动伤害切换模式
根据角色受到的伤害自动切换模组

### 6. 工具箱模式
- **轮廓工具**: 设置或移除模型轮廓
- **颜色工具**: 修改顶点颜色
- **透明度工具**: 为模型部件设置透明度

## 重要提示

⚠️ **必须确保 `GIMI-Tools` 文件夹与 `GIMI-Blender-Plugin` 在同一目录下**

插件需要调用 `GIMI-Tools` 中的 Python 脚本来执行实际功能。

## 常见问题

**Q: 插件加载失败？**
A: 检查 Blender 版本是否为 3.6.0 或更高，确认所有文件都完整。

**Q: 操作符执行报错？**
A: 打开 Blender 的系统控制台（窗口类型 > 系统控制台）查看详细错误信息。

**Q: 找不到 GIMI Tools 标签？**
A: 确保在 3D 视图窗口中按 N 键打开侧边栏，插件应该在那里显示。
